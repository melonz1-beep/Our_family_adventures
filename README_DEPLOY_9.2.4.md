# Our Family Adventures 9.2.4

Built from Version 9.2.3.

## Added
- Export Trip PDF button on each dedicated trip page.
- PDF includes trip overview, attendees, RSVP, itinerary, reservations, meals, grocery list, packing list, budget total, travel links, and memory titles/details.
- Export respects the active privacy blocker and only includes information the signed-in user is authorized to view.
- Uses jsPDF when available and falls back to the device Print / Save as PDF screen.

## Deploy
Replace all repository files with this package. Open the website once with `?v=9.2.4`, refresh, then reopen the installed app.
