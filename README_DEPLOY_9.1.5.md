# Our Family Adventures 9.1.5

## Scrapbook Studio selection repair

This release fixes the editor closing when a photo is tapped, moved, resized, or rotated. Firebase draft synchronization no longer rebuilds the scrapbook page while the dedicated editor is open. The editor remains open and the selected object remains available for editing.

## Deploy

1. Delete the old repository files.
2. Upload every file and folder from this package to the repository root.
3. Commit the changes and wait for GitHub Pages to finish.
4. Open the site with `?v=9.1.5` once to bypass the previous cache.
