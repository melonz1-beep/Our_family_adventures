# User Guide — Version 5.0

Use the bottom dock for Home, Trips, Dashboard, Memories, Scrapbook, and Chat. Use the side menu for People, Meals, Packing, Travel Links, Explore, Settings, and Our Story.

## Main features
- Home dashboard with next adventure countdown.
- People profiles with blank name field, photo saving, email, phone, and role.
- Adventures with private/family visibility, invitees, notes, weather notes, and RSVP.
- Adventure Dashboard with meals, packing, grocery list, weather, and recent memories.
- Memories 2.0 with multiple photos/videos, journal entry, date, trip, and uploader.
- Scrapbook Studio with multiple selected photos per page, layouts, themes, stickers, print/PDF.
- Meals and Grocery assigned to trips with volunteers.
- Packing lists assigned to trips.
- Family Chat with @mentions.
- Data backup download and browser-data migration.


## Version 5.0.7 Scrapbook Studio

This update adds working scrapbook layouts: Grid, Feature Photo, Collage, and Freeform. Select multiple photos from Memories, create a page, then drag photos, stickers, and text boxes around the page. Use the gold corner handle to resize. Each page supports frames, backgrounds, movable stickers, movable text boxes, JPEG export, and Print / Save as PDF.

After uploading to GitHub, open the app with `?v=5.0.7` to clear the old cache.

## Version 5.0.9 Scrapbook Draft Workflow

Scrapbook pages now save photos by reference instead of duplicating large image data into each page. This helps prevent pages from losing photos or failing to save on phones with limited browser storage.

To create a scrapbook page:
1. Open Scrapbook Studio.
2. Select one or more photos from Memories.
3. Tap **Create Draft Page**.
4. Rearrange and resize photos, stickers, or text on the draft page.
5. Tap **Create Scrapbook Page** to save the final page.
