# Our Family Adventures 9.1.6

This repair keeps Scrapbook Studio isolated from the app menu, adds selected-photo or all-photo frame controls, restores the locked Admin page entry, improves device notification testing and Firebase notification delivery, labels private trips, enforces private-trip visibility for all connected content, and allows Media uploads without a trip tag.

Deploy every file and folder from this package to the repository root. After GitHub Pages finishes, open the site once with `?v=9.1.6`.

For phone notifications, open Settings, tap Enable Notifications, then Send Test Notification. Android Chrome can display notifications while the browser or installed PWA is running. Reliable background notifications while the app is fully closed require Firebase Cloud Messaging and a server or Cloud Function.
