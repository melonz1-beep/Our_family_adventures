# Our Family Adventures — Version 6.5.4

## Corrective additions

### Family Trip History Map
- Restores the missing trip history / places visited area.
- Shows a Google map card in Explore and Dashboard.
- Uses saved adventure destinations plus manually added pins.
- Each pin has View Map and Directions buttons that open Google Maps.
- Manual pins save to the shared family data so other users can see them after login.

### Meal Planning by Trip Dates
- Adds a Trip Date Meal Planner.
- Generates meal slots from each trip's arrival/departure dates.
- Example: Friday lunch, Friday dinner, Saturday breakfast, Saturday lunch, Saturday dinner, Sunday breakfast, Sunday lunch.
- Keeps meal type instead of converting everything to Dinner.
- Allows meals to be added, assigned, and removed from the date-based planner.

### Cache/versioning
- Adds 6.5.4 CSS/JS and updates cache-busting references.
