# Our Family Adventures 10.3.5

Professional scrapbook themes, durable photo drafts, and faster on-canvas editing.

## Included upgrades

- Rebuilds every theme with coordinated paper textures, stitching, ribbons, mats, and layered embellishments.
- Gives Sunset Glow a true amber sunset, Winter Wonderland richer contrast, and Chalkboard Memories a textured slate-and-kraft scrapbook treatment.
- Adds tailored wedding, baby, pet, happy-hour, camping, and Christmas embellishments.
- Stores draft photos in IndexedDB so several photos no longer overflow localStorage and disappear after leaving the screen.
- Adds Save Draft and Finalize actions with visible confirmation.
- Changes the untitled name to placeholder text and automatically selects an existing title when tapped.
- Adds an on-canvas quick editor for photo zoom, fit/fill, frame selection, sticker sizing, rotation, duplication, and deletion.
- Adds red delete and blue resize handles directly to selected objects.
- Changes the mobile precision editor from a side panel to a bottom sheet so the scrapbook remains visible.

## Cache

Service worker cache: `ofa-10-3-5`
