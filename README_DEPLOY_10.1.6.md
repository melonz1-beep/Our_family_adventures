# Our Family Adventures 10.1.6

## Fixes
- Enlarged heart, star, flower, scalloped, oval, and speech-bubble photo frames so photos are not cut down to tiny centers.
- Shaped photos now fill the full frame while the border overlays the edge.
- Drafts autosave on edits, page changes, app backgrounding, and closing.
- Added Save Draft and Continue Editing controls.
- Full-page scrapbook viewer scales the complete portrait or landscape page into the available screen area.

## Deploy
Replace all repository files with this package and commit. Open the site with `?v=10.1.6`.

No Firebase rules or Cloud Function update is required.
