# Our Family Adventures 5.0 Installation

1. Download and unzip the package.
2. Upload all files and folders to the root of your GitHub repository.
3. Confirm these folders exist: `assets/` and `icons/`.
4. Open GitHub Pages and publish from the main branch/root folder.
5. Visit your page with `?v=50` at the end to refresh old cached files.
6. On iPhone: open in Safari, Share, Add to Home Screen.
7. On Android: open in Chrome, menu, Add to Home screen.

For Firebase production mode, paste your Firebase web app config into `firebase-config.js`, deploy the Firestore and Storage rules, enable Authentication, and enable App Check before sharing publicly.
