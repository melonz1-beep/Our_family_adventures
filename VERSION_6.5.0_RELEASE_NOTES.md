# Our Family Adventures — Version 6.5.0

This release adds the final stability layer for Firebase Authentication, profile synchronization, user-specific notifications, live weather, Google Maps cards, one floating chat button, cache cleanup, and mobile safe-area layout fixes.

## Updated files
- index.html
- app.js
- ofa-6.5.0.js
- style.css
- ofa-6.5.0.css
- service-worker.js
- manifest.json

## Deploy note
Upload the full contents of this zip to GitHub/Firebase Hosting. After deploying, open the app once in Chrome and refresh twice or reinstall the home-screen app so the new `ofa-6-5-0-core` service worker replaces older caches.
