# Version 6.6.2

Fixes Travel Links not saving/loading after the 6.6.1 click-prevention patch.

- Travel Links now save even when the page previously refreshed or swallowed the button click.
- Travel Links page loads links saved inside trips and links saved from the Travel Links tab.
- Duplicate trip links are merged into one visible link.
- Links are saved to the shared Firebase data object and local backup.
