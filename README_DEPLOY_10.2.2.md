# Our Family Adventures 10.2.2

## Fixes
- Scrapbook photo can be enlarged, shrunk, centered, and dragged inside shaped frames without moving the frame.
- The selected photo remains visibly selected while using editing controls.
- New scrapbook pages no longer reuse or create unrelated blank drafts. A draft is created only after meaningful content is added. Continue Editing tracks the exact draft selected.
- Added higher-quality illustrated Sea Glass, Baby Keepsake, Mountain Escape, Lighthouse Coast, and Camping Under the Stars backgrounds.
- Header, menu access, and bottom navigation remain available on normal app and Admin pages.
- Trip voting link cards automatically attempt to show a webpage screenshot, with the site icon as fallback. A custom thumbnail URL still overrides the automatic preview.

No Firebase rules or Cloud Function update is required.
