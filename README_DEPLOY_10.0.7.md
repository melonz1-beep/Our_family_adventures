# Our Family Adventures 10.0.7

## Included
- Chat opens in a separate browser window/tab using chat.html.
- Complete Trip moves a trip to History and displays its saved destination pin.
- Completed trips can be reopened by authorized users.
- Expanded scrapbook sticker library.
- Scrapbook editor remains open during Firebase synchronization and photo additions.

## Deploy
Replace all GitHub repository files and open `?v=10.0.7`.
No Firebase rules or Cloud Function update is required.
