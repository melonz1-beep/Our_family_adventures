# Our Family Adventures 9.0.2

This corrective rebuild updates the 9.0.1 package with:

- Always-visible People checkboxes on saved Meal, Grocery, and Packing items.
- All selected memory photos displayed and upload-result feedback when any file fails.
- Mobile-friendly destination-based live weather cards on trip and Weather pages.
- Dedicated trip modules showing actual saved content previews instead of counts only.
- Fixed-size scrapbook canvas that no longer pans or moves while editing.
- Thumbnail photo picker.
- Saved scrapbook pages rendered from exact object positions, sizes, rotation, layering, frames, stickers, emojis, and text.
- Heart, scalloped, flower, dotted, polaroid, soft, border, and no-frame options.
- Frame-color picker applied to the selected photo and preserved after save.

Deploy all files at the package root. Keep `firebase-config.js` and Firebase Authentication, Realtime Database, and Storage enabled for shared synchronization.
