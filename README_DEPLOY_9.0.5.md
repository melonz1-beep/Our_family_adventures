# Our Family Adventures 9.0.5

## Scrapbook improvements
- Full-screen pop-out scrapbook editor so the page and objects cannot scroll over app menus.
- Portrait and landscape page orientations.
- Photo repositioning inside frames and cutouts using **Adjust Photo Inside Frame**.
- Independent frame thickness, color, frame style, and cutout controls.
- New vintage, film-strip, comic, and expression-bubble frames.
- New plaid, scroll, paisley, birthday, pet, and baby backgrounds.
- New star and circle cutouts, photo-strip and mosaic layouts.
- Required page name and optional description.
- Dedicated trip scrapbook names link directly to the saved page.

## Deploy
Replace every file in the GitHub Pages repository with this package, commit, and wait for Pages to publish. Close all old app/browser windows once so service worker cache 9.0.5 activates.
