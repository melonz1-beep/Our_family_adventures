# Our Family Adventures 10.2.1

## Scrapbook draft correction

- Opening and closing a blank scrapbook editor no longer creates a draft.
- Publishing a scrapbook page no longer creates a second blank draft.
- A draft is created only after a title, description, photo, text, sticker, or other page object is added.
- Existing blank drafts may be deleted with the existing Delete draft button.

## Deploy
Replace all repository files with this package and refresh with `?v=10.2.1`.
No Firebase rules or Cloud Functions changes are required.
