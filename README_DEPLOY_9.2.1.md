# Our Family Adventures 9.2.1

Targeted fixes added on top of 9.2.0:

- Scrapbook editor remains open while Firebase synchronizes.
- Scrapbook title and description auto-save while typing.
- Adding photos no longer causes the editor to refresh or discard text.
- Separate chat opens without navigating the installed app away.
- Chat Close returns to the main app when Android will not permit a script to close the tab.
- Notification matching recognizes the signed-in user by UID, email, profile ID, owner, and name.
- Service-worker/cache version updated to 9.2.1.

Upload every file and folder to the repository root and replace the existing deployment.
