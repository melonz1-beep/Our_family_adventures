# Our Family Adventures 10.0.3

## Included updates
- Notification bell opens a per-user notification center.
- Unread notifications are bold and become read when opened.
- Notifications show who made the update and its date/time, and route to the related page.
- Mark-all-read and clear affect only the signed-in user.
- Light mode is the default; dark mode remains optional.
- The primary administrator can see all private trips while privacy remains enforced for other users.
- Group Text/Email includes Select all users.
- Chat is displayed as a chronological conversation.
- Meals, groceries, and packing are consolidated under Lists by Trip.
- The trip-level tab bar and bottom dock are removed to simplify navigation.
- Tapping outside the open sidebar closes it.

## Deployment
Replace all GitHub Pages files with this package and open `?v=10.0.3`. No Firebase rules or Cloud Function redeployment is required for this UI/navigation release.
