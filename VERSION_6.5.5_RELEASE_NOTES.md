# Version 6.5.5 Release Notes

## Added / Fixed
- Meal planning now builds meal slots from each trip arrival/departure date.
- First travel day suggests lunch/dinner; final travel day suggests breakfast/lunch.
- Family chat now saves to the shared family Firestore document so invited members can see each other's updates.
- Chat includes unread badge behavior, Mark Chat as Read button, read receipts with who read each message and what time, and retained chat history.
- Scrapbook stickers can now be moved and resized on the page, with positions saved.
- Live weather has a stronger city/state fallback and clearer destination error.
- Trips now include quick Google Maps searches for restaurants, grocery stores, gas stations, hospitals, and hotels near the selected destination.

## Deploy
Upload all files in this package to GitHub/Firebase Hosting, then clear the installed app cache by closing and reopening the app.
