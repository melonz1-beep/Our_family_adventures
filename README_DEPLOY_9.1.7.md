# Our Family Adventures 9.1.7

## Repairs
- Raises the slide-out menu above every scrapbook layer and temporarily hides scrapbook canvases while the menu is open.
- Replaces large white chat cards with compact message bubbles and a small trash icon for admins.
- Adds **Open Chat in Its Own Window**. The separate browser window/tab can remain open after the main app page is closed, as long as the browser remains running.
- Updates the application and service-worker cache to Version 9.1.7.

## Deploy
Replace all repository files with this package, commit, wait for GitHub Pages, then open the site with `?v=9.1.7`. Clear site data or uninstall/reinstall the PWA if the header still shows an older version.
