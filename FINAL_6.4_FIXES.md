# Our Family Adventures 6.4 Final Fixes

This package adds the requested 6.4 production fixes:

- New lighthouse splash/home image supplied by Melissa.
- Section navigation opens directly to the selected page instead of showing the home hero first.
- Dashboard count cards are clickable.
- Voting supports multiple links as separate voteable options.
- Vote / Change button resets the user's prior vote and saves the new vote.
- Private/group meals, groceries, and packing lists are separated.
- Private items only display for the owner.
- Group items include an assigned-to dropdown populated from app people plus Volunteer needed and Everyone / Group.
- Items can be edited, deleted, assigned, and checked off.
- Notification center with unread count, mark all read, clear all, and app badge support where Android/Chrome allows it.
- Floating draggable chat button and pop-out chat panel.
- Profile editor with photo, name, email, phone, and login button.
- Scrapbook pages render photos in themed frames with drag-to-rearrange and theme stickers for beach, mountain, ATV, fishing, party, and drink themes.
- Live weather refresh uses Open-Meteo geocoding/weather when the location can be found.
- Map helper remains linked to Google Maps for trip locations.

After uploading, uninstall the old home-screen app or clear site data so Chrome loads the updated service worker and cache.
