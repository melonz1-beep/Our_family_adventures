# Project Lighthouse — Build 0009 Legacy Core

Upload all files and folders in this directory to the root of the GitHub repository.

## Included
- Single hamburger menu
- Scrollable home cards, no overlap
- Crisp splash text and photo hero
- Memories photo/video upload for testing
- Person tags with removable tags
- Profile photos
- Working local family chat with @mentions
- Public/private/custom chapters
- Legacy timeline
- Offline-friendly service worker

Note: Build 0009 stores data in the browser with localStorage for testing. Firebase config is included in firebase.js for future live sync.
