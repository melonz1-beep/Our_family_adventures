# Security Checklist — Version 5.0

- Firebase Authentication before public family launch.
- Email verification required.
- Invite-only family access.
- Firestore rules for private adventures and roles.
- Storage rules limiting uploads to signed-in users and safe file types.
- App Check enabled before sharing widely.
- Activity log for important changes.
- Automatic backup workflow recommended.
- Never place private API admin keys in frontend code.
