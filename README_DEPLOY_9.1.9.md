# Our Family Adventures 9.2.0

Built directly from 9.1.8/9.1.7.

Fixes:
- Saved scrapbook pages are scaled and clipped to the phone screen.
- Scrapbook objects cannot paint over the app header, dock, or other cards.
- The floating chat button opens the dedicated Chat page instead of a browser popup.
- Notification permission, local test alerts, Firebase Messaging token registration, and foreground notification handling were added.
- Settings notification text contrast improved.
- Cache and manifest bumped to 9.2.0.

Deploy every file and folder to the repository root. After GitHub Pages finishes, open the website once and refresh. For remote push while the app is closed, Firebase Console must have Cloud Messaging enabled and a Web Push certificate.
