# Our Family Adventures 10.1.2

## Deploy
Replace all files in the GitHub Pages repository with this package, commit, and open the app with `?v=10.1.2`.

## Changes
- Removed visible lock icons and per-menu editing locks.
- Saved content remains editable; Delete appears only while Admin is unlocked.
- Media and Memories now show all authorized photos as thumbnails.
- Tapping a photo opens a full-size viewer with Close, Download, and Print.
- Published scrapbook pages display a preview and View, Download PDF, and Print controls.
- Floating chat opens the separate browser chat page directly.
- Admin Storage & Archive Center includes the requested Archive Status legend.

No Firebase rules or Cloud Function update is required.
