# Our Family Adventures 10.3.3

Scrapbook Studio mobile positioning and controls correction.

## Included fixes

- Centers the scaled scrapbook inside a dedicated viewport instead of positioning the unscaled 900-pixel page off-screen.
- Keeps the complete scrapbook visible after phone rotation and viewport resizing.
- Allows only Add or Edit to be open at one time so the panels do not overlap.
- Frame buttons now apply to the selected photo instead of opening the phone photo picker.
- Theme selection changes the page background and returns to the canvas.
- New photos start without an automatic Polaroid frame.
- Layers are labeled Photo 1, Photo 2, and so on instead of labeling every photo “polaroid.”
- Adds No Frame as a frame choice.

## Cache

Service worker cache: `ofa-10-3-3`

After deployment, close every app tab, open `?v=10.3.3` in the browser, and then reopen or reinstall the home-screen app.
