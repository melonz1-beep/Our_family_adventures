# Our Family Adventures — Final Deploy Documents

Use **Our-Family-Adventures-Version-5.0-Firebase-Invite-Only-5.0.11.zip** as the base project. The later 6.x ZIP files are smaller starter builds and are missing many required features.

These documents are the deployment and build handoff notes for the next final app update. They do not replace the app source code.

## Required outcome
The deployed app should include:
- Firebase Authentication enabled
- Firestore security rules deployed
- Firebase Storage rules deployed
- Real lighthouse PNG photos, not placeholder SVG icons
- Hidden admin PIN entry, not visible on screen
- Working profiles, trips, voting, memories, scrapbook, lists, meals, notes, chat, notifications, maps, and PWA install behavior

Use:
`Our-Family-Adventures-Version-5.0-Firebase-Invite-Only-5.0.11.zip`
