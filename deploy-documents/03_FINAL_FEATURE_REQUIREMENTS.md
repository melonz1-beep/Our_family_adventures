# Final Feature Requirements

## Admin
- Admin PIN must be hidden.
- Show an unlock/admin button, not the PIN value.
- PIN remains 1218.
- Activity log should not show Firebase system actions.

## Profiles
Profiles must include:
- name
- profile photo
- phone number
- email
- edit option
- delete option
- bulk select for deleting or editing multiple profiles
- option to change profile photo
- remove the fixed “family” tag from new profiles

## Trips
Trips must include:
- add trip
- edit trip
- delete trip
- voting button
- public/private trip option
- invited people selector
- private trips visible only to invited people
- no hardcoded Lake Anna / May 2027 text
- countdown by trip
- notes section on trip page
- live weather by trip area
- reservation links for Airbnb, VRBO, hotels, campgrounds, etc.
- links must open correctly

## Voting
Voting must support:
- multiple trip options
- multiple lodging options
- multiple date options
- meal voting options
- show who voted
- analytics for highest and lowest votes

## Memories and Scrapbook
Must include:
- full-resolution photo uploads
- downloadable original photos
- true scrapbook categories
- printable scrapbook pages
- true drag-and-drop layouts
- photo collage builder
- resize, rearrange, rotate, and delete collage items
- export/print option

## Lists and Meals
Must include:
- group shopping list
- individual shopping lists
- group packing list
- individual packing lists by person
- meal planning
- checkbox for people to volunteer/accept bringing a menu item

## Communication
Must include:
- group chat
- direct/individual chat
- select all people or specific people
- email option for all or selected people
- text/SMS option for all or selected people where supported by device/browser

## Notifications
Must include:
- notifications area
- opt-in for notifications
- trip reminders
- checklist reminders
- new message notifications, where supported

## Maps
Must include:
- live searchable map around trip area
- activities search
- gas search
- restaurants search
- grocery stores search
- routing/directions
- saved pins
- history map with pins for places visited

## PWA
Must work on:
- Android Chrome
- iOS Safari add-to-home-screen

Install button must not disappear unless the app is already installed or the device/browser does not support the install prompt.

## Navigation
- Side menu must scroll on phones.
- All sections must remain accessible on mobile.
