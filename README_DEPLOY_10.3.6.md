# Our Family Adventures 10.3.6

Full-page exports and a saved-page draft library for Scrapbook Studio.

## Included fixes

- Exports a clean, fixed 900 × 675 clone of the scrapbook instead of the scaled mobile editor, so JPEG and PDF include the entire page.
- Lets the user choose JPEG or PDF separately and removes selection controls from the finished file.
- Adds Close Page to desktop and mobile toolbars; closing saves the page before showing the page library.
- Adds Saved Scrapbook Pages with New Page, Open, Return, and Back to App actions.
- Stores an index of separate draft pages while keeping each page's photos in IndexedDB.
- Keeps the 10.3.5 professional themes, direct object editing, explicit drafts, and finalization workflow.
