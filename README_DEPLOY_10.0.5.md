# Our Family Adventures 10.0.5

## Included

This is the consolidated Version 10.0.5 build. It retains all updates through 10.0.4 and restores the fixed bottom navigation dock.

Bottom dock: Home, Login, Trips, Media, Scrapbook, and Lists. The selected page is highlighted, the dock supports Android and iPhone safe areas, and it remains hidden while the dedicated scrapbook editor is open. Chat remains available from the floating chat button and sidebar.

## GitHub Pages deployment

Replace the existing repository files with all files from this package. Commit the changes, wait for GitHub Pages to publish, then open `?v=10.0.5`. Close and reopen the installed app after the first refresh.

## Firebase

No Firebase rules or Cloud Function deployment is required solely for restoring the dock. The existing notification sender and scheduled seven-day reminder function remain included in the package.
