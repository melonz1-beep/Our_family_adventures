# Our Family Adventures 10.0.2

## Five-file upload limit

Media uploads and Memory uploads are limited to five files at a time. Scrapbook pages are limited to five photos total. The app displays a clear message and prevents extra selections instead of beginning an oversized upload.

Replace all GitHub Pages files with this package and open `?v=10.0.2`. No Firebase rules or Cloud Function redeployment is required for this upload-limit update.
