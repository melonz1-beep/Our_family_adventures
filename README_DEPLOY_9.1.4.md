# Our Family Adventures 9.1.4

This repair converts Scrapbook Studio into a true dedicated editing workspace on phones and tablets.

## Fixes
- The canvas is fitted into the upper workspace instead of rendering at full 680×880 size.
- The controls occupy their own independently scrollable lower panel.
- The canvas and menus never overlap.
- Touch dragging and resizing account for the fitted canvas scale.
- Drafts still save automatically when closing or leaving the editor.

## Deploy
Upload every file and folder from this package to the repository root, replacing the prior version. Wait for GitHub Pages to finish, then open the site with `?v=9.1.4` once.
