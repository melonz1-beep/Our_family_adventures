# Our Family Adventures 10.1.1

## Included
- Admin-unlock protection for Delete, Complete Trip, and Return to Active controls.
- Completing a trip preserves all trip details and media; Admin can restore it to Active.
- Improved dark-theme contrast on Admin and Archive pages.
- Working Admin navigation buttons.
- Setup Wizard and Firebase information moved to the bottom of Admin.
- Activity entries show the person, action, date, and time.
- Email address and role are hidden from the app header.
- Storage dashboard recalculates recorded usage, illustrative cost, 30-day archive readiness, and last local backup.
- Scrapbook finalized-page links remain clickable and refresh after save.

## Deploy
Replace all repository files with this package. Open the app with `?v=10.1.1`.

No Firebase rules or Cloud Function update is required for this release.
