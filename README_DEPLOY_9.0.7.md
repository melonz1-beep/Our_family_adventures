# Our Family Adventures 9.0.7

This build deepens the light palette, improves dark scrapbook-editor contrast, contains the scrapbook canvas below its controls, adds portrait/landscape/square formatting for individual photos, and makes selected photos removable while additional photos remain available from the thumbnail picker. It also adds a monochrome notification status icon and refreshes the PWA cache.

Android app-icon dots/badges require the app to be installed, notification permission to be granted, and notification badges enabled for the browser/PWA in Android launcher settings.
