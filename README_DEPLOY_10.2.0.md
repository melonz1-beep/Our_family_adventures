# Our Family Adventures 10.2.0

## Fixes
- Scrapbook pages may contain any number of photos already stored in the app.
- Media and Memories still limit each new upload to five photos.
- Finalized scrapbook cards and the full-page viewer render the complete portrait or landscape page.
- Bottom navigation remains visible and horizontally scrollable on every app page.
- New unique service-worker cache forces the complete 10.2.0 files to replace older cached versions.

## Deploy
Replace every repository file with this package, commit, then open `?v=10.2.0`. If an installed PWA still shows an older version, close it completely and reopen once.
